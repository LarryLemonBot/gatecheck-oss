# OSS Release Audit

Date: 2026-06-01

This repository was prepared as a GateCheck-only open-source release from a
bounded source extraction. It intentionally excludes the private LarryBuildsAI
monorepo, local runtime state, deployment metadata, caches, transcripts,
credentials, and unrelated product source.

## Included

- Read-only x402 metadata scanner.
- Unpaid x402 paid-path health probe.
- Deterministic receipt generation with secret-like evidence rejection.
- Agent-tool readiness checker.
- x402 launch-pack generator.
- GateCheck MCP JSON-RPC metadata and tools.
- x402 payment helper code with demo mode disabled by default.
- Focused pytest coverage and GitHub Actions CI.

## Excluded

- `.vercel/`, `.pytest_cache/`, `__pycache__/`, and generated bytecode.
- Private workspace state and local-only files.
- Non-GateCheck product source files.
- Deployment credentials, environment files, wallet secrets, and raw payment
  headers.

## Verification Commands

```bash
python -m pytest -q
rg -n -i '(api[_ -]?key|secret|token|bearer|authorization|password|private[_ -]?key|BEGIN (RSA|OPENSSH|PRIVATE)|OPENAI|ANTHROPIC|XAI|CDP|COINBASE|STRIPE|DISCORD|COOKIE|SESSION|MNEMONIC|SEED|WALLET|WEBHOOK)' . --glob '!**/.git/**' --glob '!**/__pycache__/**' --glob '!**/.pytest_cache/**' --glob '!**/.vercel/**'
```

Expected result: tests pass; scanner findings are reviewed as either generic
configuration names, public placeholders, or claim-boundary text.
